# Keystroke Dynamics Analyzer

This project explores how everyday typing behavior can be used to infer cognitive and motor patterns. It is based on the idea that keystroke timing (such as key press duration and transition delays) reflects underlying neural processes.

## 🧠 Concept

Each keystroke carries micro-level signals about how the brain coordinates movement and attention. By analyzing these signals over time, it is possible to estimate:

* Cognitive load
* Motor consistency
* Behavioral changes relative to a personal baseline

## 🔬 Approach

The system models typing behavior using:

* **Dwell Time** – duration of key presses
* **Flight Time** – delay between consecutive keys
* **Entropy** – variability in typing rhythm
* **Baseline Comparison** – detecting deviations from normal patterns

These metrics are combined to derive a composite score representing typing stability and cognitive variation.

## 💡 Motivation

The goal is to investigate whether passive interaction data (like typing) can be used as a lightweight indicator of cognitive state, without requiring specialized hardware.

## 📌 Note

This is an experimental and exploratory project intended for research and learning purposes. It does not provide medical or clinical conclusions.
